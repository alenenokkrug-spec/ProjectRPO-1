🏋️ Fitness Training Bot

Telegram-бот для тренировок, разработанный на Python. Бот помогает пользователям получать информацию об упражнениях для различных групп мышц, просматривать GIF-анимации техники выполнения упражнений и формировать удобный тренировочный процесс прямо в Telegram.

🚀 Возможности
Выбор тренировок для мужчин и женщин.
Разделение упражнений по группам мышц.
Отображение GIF-анимаций с техникой выполнения упражнений.
Удобная навигация через Inline Keyboard.
Поддержка тренировок для различных уровней подготовки.
Возможность быстрого доступа к нужному упражнению.

🎯 Цель проекта
Проект создан для упрощения доступа к тренировочным программам и демонстрации правильной техники выполнения упражнений через Telegram. Бот может использоваться как начинающими спортсменами, так и людьми, которые хотят поддерживать физическую форму.
